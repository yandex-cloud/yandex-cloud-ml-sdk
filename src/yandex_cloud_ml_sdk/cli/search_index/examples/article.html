Introduction to Vector Stores

What are Vector Stores?

Vector stores are specialized databases designed to store and search high-dimensional vector embeddings. They enable semantic search by finding documents that are conceptually similar, not just keyword matches.

Key Features

  • Semantic Search: Find documents by meaning, not just keywords
  • Automatic Chunking: Split large documents into searchable pieces
  • Multi-format Support: Process PDF, DOCX, Markdown, and more
  • Metadata Filtering: Narrow searches with custom attributes

Use Cases

Vector stores power many AI applications:

  1. Document Q&A systems
  2. Knowledge base search
  3. Recommendation engines
  4. Content discovery

Getting Started

Use the vector_stores CLI tool to create your first vector store from local files: 

vector_stores local ./docs --name "my-knowledge-base"
